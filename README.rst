Gíog
====

An ogham Twitter client. Requires Python 3.3 or later.

gíog
  'to tweet'

Status
------

There's no Twitter code yet, but there is some rudimentary, possibly erroneous
ASCII |->| Ogham conversion.

.. |->| replace:: →

Try this::

  $ python3
  >>> import giog
  >>> giog.ogham.test('This is a test!')
  this is a test
  ᚛ᚈᚆᚔᚄ ᚔᚄ ᚐ ᚈᚓᚄᚈ᚜
