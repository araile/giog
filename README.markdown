gíog
====

An Ogham Twitter client, just for fun.

_gíog ≈ to tweet_


Status
------

There's no Twitter code yet, but there is some rudimentary, possibly erroneous
ASCII &rarr; Ogham conversion.

Try this:

    >>> import ogham
    >>> ogham.test('this is a test')
    this is a test
    ᚛ᚈᚆᚔᚄ ᚔᚄ ᚐ ᚈᚓᚄᚈ᚜
